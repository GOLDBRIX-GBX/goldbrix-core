GoldBrix Core release notes
===========================

v30-gbx (2026)
--------------

First GoldBrix Core release line. GoldBrix is an autonomous, permissionless
proof-of-work network:

- SHA-256d proof of work; open, permissionless mining; no premine, no presale.
- Fixed maximum supply of 15,000,000 GBX.
- Emission schedule and all economic rules enforced in consensus code.
- Deep-reorg protection to raise the cost of history-rewriting attacks.
- Fully reproducible builds (GNU Guix); published SHA256SUMS let anyone rebuild
  from source and verify the binaries bit-for-bit.

License information is in COPYING.
